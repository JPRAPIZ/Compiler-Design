# Compiler-Design

How to run in Linux/Unix:

in front end do npm run dev

in backend do
source venv/bin/activate
python3 server.py

How to operate in Windows:

Frontend:
1. cd to the frontend directory
2. npm run dev

Backend:
1. cd to the backend directory
2. .\venv\Scripts\Activate
3. python server.py

To get out of the (venv) environment, type deactivate in the terminal.
